RUUT 기술 문서
=======================================

.. rst-class:: table-width-fix

SKT 의 도로 교통 정보 플랫폼인 RUUT 기술 문서를 방문해 주신 것을 환영 합니다. RUUT 는 모빌리티 서비스에 필요한 다양한 동적 교통 데이터 (Intelligent Road Data, IRD)를 특정 지도에 무관한 형태로 변환한 후 표준 API 형태로 외부에 제공하는 플랫폼 입니다.

문서 내 도움이 필요하실 경우 admin_ruut@sktelecom.com 으로 연락 주시기 바랍니다.

--------------------------

* 로그인을 통한 인증 토큰 획득
